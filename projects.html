Ballroom Competition Manager (GitHub)

SQL Interpreter

I built with two other students from scratch a SQL interpreter in Java that takes in a set of data files and then efficiently processes and evaluates SQL queries on that database. We expanded and addressed shortcomings by supporting multiple physical implementations for relational operators, three join plans, B+-tree indexing, and the gathering of statistics for informed optimization.


OCaml REPL Interpreter

I implemented evaluation and type inference, as well as a test suite, for an interpreter of a functional language containing many of the core features of OCaml, with a big-step environment model semantics.


Ranking Manager (GitHub)

Upon being provided inputs from the user, the ranking manager waits for user inputs to decide how to add to a list of rankings. Assuming strict preferences, the program efficiently adds elements to the ranking list in the correct order in a systemic way.


Tournament Emulator (GitHub)

The tournament emulator simulates a tournament based on the comparative percentages of winning among the other competitors. Based on a seeding system, the program simulates a bracket tournament. After displaying the winner of each match-up, the emulator picks a winner and returns a full ranking.
